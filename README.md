### Centos 7.x / Mac OS
```shell
# 检查Nginx配置的正确性
nginx -t

# 启动Nginx服务
nginx

# 停止Nginx服务
nginx -s stop

# 重载Nginx配置
nginx -s reload

# 重新打开日志文件(让Nginx把新日志写入新文件中，以主动完成日志切割)
nginx -s reopen

# 查看Nginx版本号
nginx -v

# 查看Nginx帮助
nginx -h

# 查看Nginx进程
ps -ef | grep nginx 
```
